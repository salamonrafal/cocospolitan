## Instalation
### Clone from GIT

```bash
GIT clone https://github.com/salamonrafal/cocospolitan.git
```

### Install packages

```bash 
npm i
```

### Build 

```bash
npm run build:production
```

### Rest API for development

```bash
npm run start:api
```
